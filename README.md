Repositório com os exercícios e projetos desenvolvidos durante a Piscine da 42.

## Sobre

A Piscine é um programa intensivo de aprendizagem que introduz os fundamentos da programação, resolução de problemas e trabalho autónomo através de projetos práticos e avaliações entre pares.

Este repositório reúne os trabalhos realizados ao longo da minha participação na Piscine.

## Conteúdo

- Shell (Shell00, Shell01)
- C (C00 a C06)
- Rushes
- 
## Objetivos

- Desenvolver lógica de programação
- Aprender os fundamentos da linguagem C
- Melhorar a capacidade de resolução de problemas
- Praticar trabalho colaborativo e peer learning
